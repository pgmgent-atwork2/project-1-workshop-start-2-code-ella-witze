Hier gaan we verandering in brengen door de CSS aan te passen. Dit is het toevoegen van styling aan je site.   -       STAP 1:   Ga naar het mapje css waarin je een file vindt genaamd style.css. Hierin zoek je naar: /* GRID */ Hieronder zal je de code schrijven.   Om je code te linken aan je html zal je de klasses die je hiervoor hebt gebruikt weer aanspreken.   In je css file vind je vanboven :root. Hierin staan al een paar voorgecodeerde kleuren die je zal mogen gebruiken bij je styling. Dit doe je door een punt te plaatsen en direct daarachter de naam van je klasse.   Je begint met je kleine container:   .grid {             display: grid;             width: 30%             grid-template-columns: 1fr 1fr 1fr 1fr;             grid-template-rows: 1fr 1fr 1fr 1fr; border: solid 1rem var(--darkgreen-color); // kleur uit de :root }   Nu doe je de grote container dit plaats je onder de grid code na het gesloten haakje:   .grid-wrap {             Margin-top: 5rem;             Width: 100vh;             Display: flex;             Gap: 20rem;             Flex-wrap: wrap;             Justify-content: center;             Align-items: center; }   Met de display flex kan je de positie van de kleinen container bepalen. Door justify content verander je de verticale positie en door align-items de horizontale positie, deze zetten we allebei op het midden.   Als je nu kijkt op je site heb je een mooi grid waar je bloemen, graan, etc. mooi in het midden staat.

On the home screen, click squares to reveal quiz questions. Answer correctly to start a timer and grow crops. Harvest by recognizing a green background, then sell in your inventory. Use money to buy fertilizer in the Market so your crops grow faster!
